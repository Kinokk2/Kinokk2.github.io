## ⚡ Speed Tips

**Fastest way to customize:**
1. Use Find & Replace (Cmd/Ctrl + Shift + F in Cursor)
2. Search for "YOUR NAME" → Replace with your actual name
3. Search for "your.email@example.com" → Replace with your email
4. Search for "Project Title" → Replace one by one with your projects

**Image optimization:**
- Use tinypng.com to compress images
- Resize to appropriate dimensions before uploading
- Convert to WebP for better performance (optional)

## 🆘 Stuck? Common Issues

**Can't see changes?**
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

**Images not showing?**
- Check file path is correct
- File names are case-sensitive
- Make sure image is in /images folder

**Colors not changing?**
- Edit CSS variables in styles.css
- Make sure you're editing the right file
- Clear browser cache

**Mobile menu not working?**
- Make sure script.js is linked
- Check browser console for errors (F12)
