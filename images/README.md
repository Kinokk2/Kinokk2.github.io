# Images Folder

## Required Images:

### Navigation Avatar
- **Filename:** `kelvin-avatar.jpg`
- **Size:** 200x200px (square)
- **Format:** JPG or PNG
- **Usage:** Your profile picture in the navigation bar

### Project Covers (Homepage)
- **usps-cover.jpg** - USPS project thumbnail (1600x1000px)
- **pinterest-cover.jpg** - Pinterest project thumbnail (1600x1000px)

### About Page
- **your-photo.jpg** - Full headshot for About page (600x800px, 3:4 ratio)

### Case Study Images
Place in this folder or create subfolders like:
- `/images/usps/` - All USPS redesign images
- `/images/pinterest/` - All Pinterest redesign images

## Image Tips:
- Compress all images before uploading (use tinypng.com)
- Use WebP format for better performance (optional)
- Keep filenames lowercase with hyphens (e.g., `project-hero.jpg`)

## Placeholder Avatar:
Until you add your real avatar, a placeholder initial will show in the nav.
